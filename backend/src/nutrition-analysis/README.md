# Nutrition Calculation – Scientific Validation (Short Reference)

Logika perhitungan nutrisi berikut mengacu pada standar internasional untuk memastikan akurasi dan konsistensi.

## Referensi Utama

* **WHO – Nutrient Requirements & RDA**: [https://www.who.int/publications](https://www.who.int/publications)
* **EFSA Dietary Reference Values**: [https://www.efsa.europa.eu/en/topics/topic/dietary-reference-values](https://www.efsa.europa.eu/en/topics/topic/dietary-reference-values)
* **USDA FoodData Central**: [https://fdc.nal.usda.gov](https://fdc.nal.usda.gov)
* **FAO Human Nutrition Guidelines**: [https://www.fao.org/nutrition](https://www.fao.org/nutrition)

## Dukungan Perhitungan

* Kalori = (Protein × 4) + (Karbohidrat × 4) + (Lemak × 9)
* Rekomendasi harian disesuaikan dengan: usia, jenis kelamin, berat/tinggi, dan aktivitas (WHO/EFSA)
* Sumber nutrisi makanan mengikuti database USDA (FoodData Central)

## Kode yang Didukung

* Perhitungan nutrisi makro/mikro
* Mapping makanan → nutrisi (FoodData Central ID / internal DB)
* Validasi RDA menggunakan nilai WHO/EFSA
